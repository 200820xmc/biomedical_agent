## 历史归档

[`archive/`](archive/)保存早期需求、自查报告、旧问题诊断和旧评测方案。归档文档可能包含Top-50、Listwise Rerank、来源多样性、69篇语料或旧数据集路径，只用于追溯，不代表当前实现。

## 维护规则

1. 当前参数只在`AGENTS.md`和当前设计文档中维护。
2. 阶段性实验结果保存在`evaluation/results/{run_id}/`，不覆盖旧run。
3. 被新实现替代但仍有追溯价值的文档移入archive。
4. 空白、重复且无追溯价值的文档才直接删除。
5. 新增文档必须加入本导航，避免出现孤立文件。
